# SIAB - Product Strategy, PRD and Functional Specification
Version: June 2026
Consolidated document for Cloud Code / Claude Code development.
# Vision
SIAB is an Environmental Intelligence and Biodiversity Compliance platform integrating camera traps, drones and satellites to automate biodiversity monitoring and environmental compliance.
# Market Strategy
Dual strategy: (1) Commercial sales to environmental consultancies executing EIA/RIMA and monitoring programs. (2) Scale and data acquisition through conservation projects, NGOs, reserves and sustainability funding.
# ICP
Environmental consultancies, biodiversity monitoring firms, licensing specialists, later expanding to regulated sectors, governments and conservation organizations.
# Personas
Fauna Analyst, Environmental Project Coordinator, Licensing Specialist, SIAB Administrator.
# Competitive Benchmark
Wildlife Insights, Zamba, NatureMetrics, Planet, Satelligence, Envirosuite, Montrose, ERM, Arcadis. SIAB differentiates through fauna + flora + drones + satellites + compliance + AI.
# Architecture
DDD, Event-Driven, Cloud Native, API First, Dataset Centric, Model Agnostic. Domains: Monitoring, Taxonomy, Analytics, Intelligence, Compliance, Remote Sensing.
# AWS Stack
S3, ECS Fargate, SQS, EventBridge, DynamoDB, CloudWatch, Terraform, FastAPI, React/Next.js, Streamlit MVP, YOLO/PyTorch.
# MVP Functionalities
Project Management, Video Upload, Automatic Registration, Frame Extraction, Fauna Detection, Evidence Persistence, Biodiversity Dashboard, Evidence Gallery, CSV Export, Human Review, Species Catalog, Intelligent Reports, Chat with Data.
# Future Domains
Compliance management, drones, satellites, flora monitoring, environmental intelligence, biodiversity digital twin.
# Roadmap
MVP: fauna monitoring. V1: consultancies and drones. V2: satellites and flora. V3: environmental intelligence and compliance platform.
# Success Metrics
Projects monitored, videos processed, hectares monitored, AI precision/recall, processing cost, recurring revenue, connected cameras.
